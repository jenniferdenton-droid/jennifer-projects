⬆ Push to GitHub

Copy and paste this into Terminal to push your latest dashboard to GitHub Pages:

cd ~/Documents/Claude/Projects && git add jennifer-todo-dashboard.html && git commit -m "Manual sync $(date)" && git push origin main

After the push completes, wait ~60 seconds then hit Cmd+Shift+R on your GitHub Pages link to see the update.

⏰ The 5pm scheduled task will also push automatically every weekday.
